Example1.Base.Networks.BaseWithPF

Partial SMIB Model with power flow data, system data block and a load.

Extended model that contains power flow data (see the pf record) and a load with an RealInput interface.

Extends from Base (Partial model containing basic network elements and SysData component).

partial model BaseWithPF "Partial SMIB Model with power flow data, system data block and a load." extends Base( B2(v_0=pf.bus.V2, angle_0=pf.bus.A2), B1(v_0=pf.bus.V1, angle_0=pf.bus.A1), B3(v_0=pf.bus.V3, angle_0=pf.bus.A3), B4(v_0=pf.bus.V4, angle_0=pf.bus.A4)); OpenIPSL.Electrical.Loads.PSSE.Load_ExtInput load( v_0=pf.bus.V4, angle_0=pf.bus.A4, P_0=pf.loads.PL1, Q_0=pf.loads.QL1, d_P=Modelica.Constants.eps, t1=Modelica.Constants.inf, d_t=Modelica.Constants.eps); Plants.inf inf( P_0=pf.machines.PG2, Q_0=pf.machines.QG2, v_0=pf.bus.V3, angle_0=pf.bus.A3); PFData.PFScenarios pf "Specify power flow data to derive star values"; equation connect(load.p, line_2.n); connect(inf.pwPin, B3.p); end BaseWithPF;
